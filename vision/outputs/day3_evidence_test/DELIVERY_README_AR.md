# حزمة أدلة الحوادث لليوم الثالث

تحتوي الحزمة على ملف `events.json`، وتعريف المناطق `zones_dataset1.json`،
وصور الحوادث ومقاطع الأدلة داخل مجلد `evidence`.

حدود المناطق مرسومة على الصور والمقاطع:

- الأخضر: مسارات واتجاهات الاقتراب.
- البرتقالي: منطقة امتداد الطابور.
- الأحمر: خط التوقف.

المسارات داخل `events.json` نسبية ومحمولة. للتجربة المحلية، شغّل من داخل
المجلد بعد فك الضغط:

```powershell
python -m http.server 8000
```

ثم يصبح ملف الحوادث متاحًا على `http://localhost:8000/events.json`، وتصبح
الصور والمقاطع متاحة تحت `http://localhost:8000/evidence/`.

هذه الحزمة مخصصة حاليًا للعرض التجريبي المعتمد على الملفات. لا تتضمن خدمة
Live Event API أو WebSocket؛ ويمكن لخادم الواجهة الخلفية خدمة المجلد نفسه
كوسائط ثابتة وتحويل المسارات النسبية إلى روابط الخادم.
